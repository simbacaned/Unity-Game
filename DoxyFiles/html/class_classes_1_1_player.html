Unit Classes.Player.GetNewUnit ( )

Get a new unit from the player's list of units.

If there are military units, return the first one found. If there are no military units, check for cities and return the first one found. If there are no cities, check for passive units and return the first one found. If there are no units, return null.

Returns
The new unit to be added to the player's list of units
